Language Reference

CoffeeScript uses significant whitespace to delimit blocks of code. Instead of using curly braces { } to surround blocks of code, use indentation. You don't need to use parentheses to invoke a function if you're passing arguments. The implicit call wraps forward to the end of the line or block expression.
